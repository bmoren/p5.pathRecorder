This example has pre-laoded paths from an external json file, click and drag to add more frames to the buffer.

press the 'c' key to clear the buffer

press the 's' key to save the buffer to an external file

press the 'p' key to show / hide the paths

press the 'r' key to reset the animation playback

playback modes:

press the 'f' key for forward playback

press the 'b' key for reverse (backwards) playback

press the 'a' key for alternating (palindrome / back-to-back loop) playback
